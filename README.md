# Инструкция по использованию Cursor Tool

## Основной алгоритм абуза (сброса и регистрации):

1.  **Сброс ID**: Нажмите кнопку **«Сбросить Cursor»** и дождитесь завершения процесса в открывшемся окне PowerShell.
2.  **Загрузка почт**: Нажмите **«Выбрать файл»** и укажите ваш `.txt` файл со списком почт (формат `login:password`).
3.  **Генерация данных**: Нажмите **«Сгенерировать один аккаунт»**. Данные для регистрации появятся в правой колонке и сохранятся в `accounts.txt`.
4.  **Регистрация**: Откройте Cursor и начните регистрацию, используя сгенерированные данные (Имя, Фамилия, Почта, Пароль).
5.  **Получение кода**: Если Cursor запрашивает код подтверждения:
    - Убедитесь, что в поле **«NotLetters API ключ»** введен ваш ключ.
    - Нажмите **«Получить последний код с почты NotLetters»**.
    - Скопируйте полученный код и вставьте его в Cursor.
6.  **Готово**: Пользуйтесь новой лицензией!

---

## Решение проблем:

### Если запрашивает номер телефона (Verification):
- Попробуйте зайти через другой браузер или создайте новый профиль в текущем браузере.
- Полностью очистите Cookie и кэш в браузере.

### Если пишет "Доступ заблокирован" (Access Blocked):
- Смените VPN (другая локация или провайдер).
- Смените почту и сгенерируйте новые данные аккаунта в программе.
- Попробуйте зайти через другой браузер/профиль или сбросьте Cookie/кэш.

---

## Техническая информация:
- **Конфигурация**: API-ключ хранится в `%LOCALAPPDATA%\CursorTool`.
- **Логи**: Все действия отображаются в окне «ЛОГ» внутри программы.
